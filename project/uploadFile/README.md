
## [通过文件地址直接下载](./nodeUpload/fileAddress)

1. 在空目录test下新建service目录模拟后端，新建index.html文件模拟前端。

2. 安装serve用来启动静态资源服务器

```js
npm install -g serve
```

3. 进入service目录，启动服务

```shell
cd service 
serve -s
```
